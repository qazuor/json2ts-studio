# JSON2TS Studio

Conversor de JSON a tipos TypeScript pensado para uso rápido: pegás un ejemplo de payload y obtenés tipos limpios y editables.

- Features
  - Generación de interfaces y types a partir de un JSON de ejemplo
  - Opciones para marcar propiedades opcionales o estrictas
  - Soporte para arrays anidados y tipos compuestos

- Stack
  - React
  - TypeScript
  - Tailwind CSS
